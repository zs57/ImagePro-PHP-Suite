# Security Policy | ImagePro Elite

Security is the cornerstone of the **ImagePro Elite** engine. We are committed to maintaining a hardened environment for all enterprise integrations.

## 🛡️ Supported Versions
| Version | Supported |
| ------- | --------- |
| 5.0.x   | ✅ Active  |
| 4.x.x   | ⚠️ Legacy  |
| < 4.0   | ❌ End of Life |

## 🚨 Reporting a Vulnerability
Please do not report security vulnerabilities via public GitHub issues. Instead, contact the **zs57** security team directly or open a private security advisory.

We aim to respond to all critical reports within 24 hours.

## ✅ Hardened Features
- Mandatory MIME-Binary sniffing via `finfo`.
- Automatic memory buffer management.
- Path traversal protection.
