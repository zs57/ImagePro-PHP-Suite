# Security Policy | ImagePro

## Reporting a Security Bug
If you discover a security vulnerability, please send an e-mail to **z51722369@gmail.com**. All security vulnerabilities will be promptly addressed.

Please do not open a public GitHub issue for security-related bugs.

## Hardened Features
- **MIME Sniffing**: Uses `finfo` binary inspection to verify image integrity.
- **Path Protection**: Standardized absolute path resolution in all factories.
- **Memory Caps**: Configurable memory limits per-instance.
