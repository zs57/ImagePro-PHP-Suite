# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-24
### Added
- **PSR-4 Support**: Full autoloading support via Composer.
- **Multi-Driver Parity**: 100% feature match between GD and Imagick.
- **Advanced API**: Added `resize()` modes, `save()` format support, and `withMemoryLimit()`.
- **Elite Features**: `autoOrient()` and `stripMetadata()`.
- **DevOps**: Integrated PHPUnit suite and GitHub Actions CI (PHP 8.1 - 8.3).
- **Architecture**: Decoupled Enums and Exceptions into separate namespaces.
