كيفية الاستخدام (Namespaced)

Usage (Namespaced)

تستخدم المكتبة مساحة العمل ImagePro\Enterprise لضمان عدم التعارض:

The library uses the ImagePro\Enterprise namespace to ensure compatibility:

use ImagePro\Enterprise\ImagePro;
use ImagePro\Enterprise\ImageFilter;

ImagePro::open('img.jpg')
    ->withMemoryLimit('256M')
    ->filter(ImageFilter::SHARPEN)
    ->save('output.webp');

أمثلة برمجية (Driver-Agnostic)

Code Examples (Driver-Agnostic)

تدعم المكتبة التنقل التلقائي بين محركي GD و Imagick:

The library supports automatic failover between GD and Imagick engines:

ImagePro::open('photo.jpg')
    ->resize(1200, 800)
    ->stripMetadata()
    ->save('output.jpg', quality: 90);
