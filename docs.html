Code Examples (Driver-Agnostic)

تدعم المكتبة التنقل التلقائي بين GD و Imagick:

The library supports automatic failover between GD and Imagick:

ImagePro::open('photo.jpg')
    ->autoOptimize()
    ->filter(ImageFilter::SEPIA)
    ->watermark('zs57 Enterprise')
    ->convertToWebP('output.webp');
