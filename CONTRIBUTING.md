# Contribution Guide | ImagePro Elite

Welcome to the **zs57 Open Source Initiative**. We are thrilled that you are considering contributing to the most resilient image optimization suite for PHP.

## 🏛️ Development Standards
- **Strict Typing**: All new code must use PHP 8.1+ strict typing.
- **Namespacing**: Adhere to the `ImagePro\Enterprise` namespace architecture.
- **Documentation**: Any new feature must be documented in `src/docs.html`.

## 🚀 How to Contribute
1. Fork the repository.
2. Create a feature branch (`git checkout -b feature/EliteInnovation`).
3. Commit your changes with professional, descriptive messages.
4. Push to the branch.
5. Open a Pull Request for review by the **zs57** core team.

## ⚖️ Code of Conduct
We foster a professional, inclusive, and elite environment. Respect and technical excellence are our core values.
